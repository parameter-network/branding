# Parameter branding assets

This repository contains the **official branding assets** of the Parameter network.

All logos, icons, and related materials provided here are the only approved visual representations of the Parameter brand and are intended for use by exchanges, wallets, developers, media, and partners.

---

## About Parameter

**Parameter** is a peer to peer AI Learning network with hybrid consensus algorithm (PoW + PoAI - Proof of AI)

This repository serves as a **single source of truth** for all official Parameter visual assets.

---

## Purpose of this Repository

The assets in this repository may be used for:

- Cryptocurrency exchanges
- Wallets and explorers
- dApps and integrations
- Articles, presentations, and media publications

Using assets from this repository helps ensure brand consistency across the ecosystem.
